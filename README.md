# PHP PROJECT - PHASE 1
# FunDee Collection
A modern fashion company created for our PHP Project which displays a collection of clothes in the trendy era for users to purchase and shop

# Technologies
- HTML
- CSS
- PHP
- Google Fonts
- MySQL
- Version Control via Github

# Project Purposes
- Homepage with carousel and feature cards as the first inroduction to the company 
- About page which contains the purpose of the company
- Shop page which previews the products available
- Individual product pages for user to explore more about the product
- Contact page for users in need of help with products
- Login and Register page for users to be able to easily shop with an account

# Setup Instructions
To view the project so far:
- Download the repository from GitHub
- Upload in a code editor eg Visual Studio Code
- View the pages on your local server (XAMPP/LAMPSTACK)
- Database included but not yet connected
  To explore the pages on your browser:
  - index.php - homepage
  - about.php - about page
  - shop.php - shop page
  - product-1.php - one of the product pages(there are 6 product pages)
  - contact.php - contact form
  - login.php/register.php - account pages
